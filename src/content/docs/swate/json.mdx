For reusing filled metadata tables, you can use the export and import function of swate. The file format used for this is JSON. Swate supports multiple different import and export formats for JSON, we recommend using ARCtrl JSON. 

## Export

<Steps>

    1. Fill out annotation table you want to reuse 
    2. Open Sidebar (little arrrow bottom right) 
    3. Go to JSON Export tab
    
        The "File Export" function can be found in the <a href={SwateOverview.src} target="_blank">Sidebar</a>.

        <Tabs syncKey="swateHost">
        <TabItem label="Browser">

        You can export your data sheet simply by clicking on "Download".

        </TabItem>
        <TabItem label="Excel">

        In contrast to the web version of swate, you cannot simply download your annotation tables.
        In order to do so, you have to create a [top level metadata sheet](/nfdi4plants.knowledgebase/swate/top-level-metadata).

        </TabItem>
        </Tabs>

        ![](@images/swate/json-export.png)
    4. Select ARCtrl and click download button
    5. Move file to folder of your choice 
    
</Steps>
