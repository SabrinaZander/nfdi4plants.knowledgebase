:::note
This guide shows you how to generate a Group Access Token.
A Group Access Token can grant access to **all ARCs** of one group.
:::

<Steps>

1. Sign in to the [DataHUB](https://git.nfdi4plants.org/)
2. Navigate to the group's settings
2. Open the `Access Tokens` settings in the sidebar `(1)`
3. Enter or adapt the required token information `(2)` (more about token information [here](/nfdi4plants.knowledgebase/datahub/access-tokens#token-information))
4. Click <btn>Create group access token</btn> `(3)`

    ![](@images/datahub/access-tokens/datahub-access-token-group-01.drawio.png)

5. Your new group access token appears on top. Copy `(4)` and use it in the desired application.

    ![](@images/datahub/access-tokens/datahub-access-token-group-02.drawio.png)

</Steps>
