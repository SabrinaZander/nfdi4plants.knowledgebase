This guide shows you how to generate a Group Access Token.

<Steps>

1. Sign in to the [DataHUB](https://git.nfdi4plants.org/)
2. Navigate to the group for which you want to create an access token
3. Navigate to 'Settings' (1), then 'Access Tokens' (2): ![Alt text](@images/datahub/datahub-access-token-project-2.png)
3. Choose a name for your token (1), set or remove the expiration date (2), assign a role (3), select the scopes (4), and create your token (5):
    ![Alt text](@images/datahub/datahub-access-token-project-3.png)
4. Once created, you can copy the token:
    ![Alt text](@images/datahub/datahub-access-token-project-4.png) 

</Steps>
