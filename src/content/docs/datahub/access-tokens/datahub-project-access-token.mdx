:::note
This guide shows you how to generate a **Project Access Token**.  
Project Access Tokens are scoped to a single ARC and cannot be used for other ARCs. 
:::

<Steps>

1. Sign in to the [DataHUB](https://git.nfdi4plants.org/)
2. Open your Project (ARC)
3. Navigate to the [ARC's settings](/nfdi4plants.knowledgebase/datahub/navigation-settings/datahub-arc-settings/)
4. Open the `Access Tokens` settings in the sidebar `(1)`
5. Enter or adapt the required token information `(2)` (more about token information [here](/nfdi4plants.knowledgebase/datahub/access-tokens#token-information))
6. Click <btn>Create project access token</btn> `(3)`

    ![](@images/datahub/access-tokens/datahub-access-token-project-01.drawio.png)

7. Your new project access token appears on top. Copy (4) and use it in the desired application.

    ![](@images/datahub/access-tokens/datahub-access-token-project-02.drawio.png)

</Steps>
